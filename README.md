# ar-nenga2024

A Three.js-based project that creates a 3D animation of "HAPPY NEW YEAR 2024" text with a glowing material and a floating floor.

## Demo
No live demo available.

## Features
- Generates 3D text "HAPPY NEW YEAR 2024" with a custom glowing material
- Animates the text with rotation and position changes
- Creates a floating floor and ceiling to enhance the 3D scene
- Includes a version with bloom post-processing effect

## Requirements
This project requires a modern web browser that supports JavaScript modules and WebGL.

## Usage
1. Clone the repository:
   ```
   git clone https://github.com/code4fukui/ar-nenga2024.git
   ```
2. Open the HTML files (`index.html`, `logo.html`, or `bloom.html`) in a web browser.

## Data / API
This project uses the following external libraries:
- [Three.js](https://threejs.org/) for 3D rendering
- [egxr.js](https://github.com/code4fukui/egxr.js) for the Three.js setup
- [createMeshIJFont.js](https://github.com/ichigojam/font) for generating 3D text
