# ar-nenga2024

3Dアニメーションで新年を祝う、Webアプリケーションです。ブラウザで手軽に楽しめます。

## 機能
- 3Dテキスト「HAPPY NEW YEAR 2024」をシェーダーアニメーションで演出
- 浮遊する床と天井で3Dシーンを演出
- ブルームエフェクトを使った発光モデルのバージョンも用意

## 必要環境
WebブラウザとWebGLに対応したデバイス。

## 使い方
ブラウザで[ar-nenga2024のデモ](https://code4fukui.github.io/ar-nenga2024/)を開くと、3Dアニメーションが再生されます。
スマートフォンでアクセスするとARモードで表示できます。
