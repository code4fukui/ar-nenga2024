# ar-nenga2024

3Dアニメーションで新年を祝う、WebARアプリケーションです。ブラウザで手軽に楽しめます。

## デモ
[ar-nenga2024のデモ](https://code4fukui.github.io/ar-nenga2024/)

## 機能
- 3D空間にテキストオブジェクトを表示
- テキストをシェーダーアニメーションで演出
- ARモードでスマートフォンの画面上に3Dオブジェクトを配置

## 必要環境
WebブラウザとWebGLに対応したデバイス。

## 使い方
ブラウザで[デモページ](https://code4fukui.github.io/ar-nenga2024/)を開くと、3Dアニメーションが再生されます。
スマートフォンでアクセスするとARモードで表示できます。
